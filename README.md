# Windows Helper
Windows Helper - это консольная утилита для очистки системы. Программа предоставляет много инструментов для разблокировки компьютера, очистки системы от мусора, удаления вирусов.

Основные возможности:
 Система навигации:
- Интуитивный интерфейс - выбор и переход между страницами осуществляется по номерам
- Многоуровневое меню - удобная организация инструментов по категориям

Основные инструменты:
 Файловые менеджеры:
- Explorer++ - расширенный файловый менеджер
- FAR Manager - консольный файловый менеджер

Утилиты разблокировки:
- SimpleUnlocker - инструмент для разблокировки файлов и процессов

Системное обслуживание:
- Очистка системы от временных файлов и мусора
- Удаление вредоносного ПО и вирусов
- Оптимизация работы системы

 Системные требования:
- Операционная система: Windows XP/7/8/10/11
- Архитектура: x86/x64
- Тип: Консольное приложение (в cmd.exe)


 Установка и запуск
 Следуйте инструкциям дальше. В конце можно поставить галочку для запуска программы, или запустить с ярлыка на рабочем столе.

 Использование:

1. После запуска программы откроется главное меню
2. Для выбора нужного раздела введите соответствующий номер
3. Для возврата в предыдущее меню используйте соответствующую команду
4. Следуйте инструкциям для каждой конкретной утилиты

Где можно использовать:
 Программа предназначена исключительно для легального использования и пользования на собственных компьютеров. Перед использованием убедитесь, что у вас есть права администратора к системе.

При возникновении проблем с работой программы:
1. Убедитесь, что программа запущена с правами администратора
2. Проверьте антивирусное ПО на ложные срабатывания
3. Убедитесь в совместимости выбранной утилиты с вашей версией Windows.
